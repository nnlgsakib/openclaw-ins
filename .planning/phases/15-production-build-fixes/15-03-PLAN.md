From src-tauri/src/commands/silent.rs:
```rust
pub const QUICK_TIMEOUT: u64 = 30;
pub const INSTALL_TIMEOUT: u64 = 120;
pub fn silent_cmd(program: &str) -> tokio::process::Command;
pub async fn run_with_timeout(cmd: &mut tokio::process::Command, timeout_secs: u64) -> Result<std::process::Output, String>;
```

**CRITICAL:** Do NOT chain methods on the result of `silent_cmd()`. Split into two statements to avoid E0716 borrow errors:
```rust
// WRONG (causes E0716):
let mut cmd = silent_cmd("node").arg("--version");
// CORRECT:
let mut cmd = silent_cmd("node");
cmd.arg("--version");
```

Migration pattern for .output().await calls:
```rust
// BEFORE:
let output = tokio::process::Command::new("npm")
    .args(["install", "-g", "openclaw@latest"])
    .output()
    .await
    .map_err(|e| ...)?;
// AFTER:
let mut cmd = silent_cmd("npm");
cmd.args(["install", "-g", "openclaw@latest"]);
let output = run_with_timeout(&mut cmd, INSTALL_TIMEOUT)
    .await
    .map_err(|e| ...)?;
```

Migration pattern for spawned streaming processes (git clone, docker pull):
```rust
// BEFORE:
let mut child = tokio::process::Command::new("git")
    .args(["clone", "--depth", "1", "--progress", ...])
    .stdout(Stdio::piped())
    .stderr(Stdio::piped())
    .spawn()...;
// AFTER:
let mut cmd = silent_cmd("git");
cmd.args(["clone", "--depth", "1", "--progress", ...]);
cmd.stdout(Stdio::piped());
cmd.stderr(Stdio::piped());
let mut child = cmd.spawn()...;
```
