<objective>
Fix 8 compilation errors (E0716: temporary value dropped while borrowed) and 1 warning in the already-migrated command files. These errors prevent the project from compiling after the initial silent_cmd migration.

Purpose: The initial migration of command files to use `silent_cmd()` introduced borrow checker errors because method chains like `silent_cmd("node").arg("--version")` create temporaries that are dropped before `run_with_timeout(&mut cmd, ...)` can borrow them. The fix is to split each chain into two statements.

Output: Clean compilation (`cargo check` exits 0).
</objective>

The bug pattern — method chain creates temporary that gets dropped:
```rust
// BROKEN: .arg() returns &mut Command (temporary), cmd borrows it
let mut cmd = silent_cmd("node").arg("--version");
let output = run_with_timeout(&mut cmd, QUICK_TIMEOUT).await;

// FIXED: split into two statements
let mut cmd = silent_cmd("node");
cmd.arg("--version");
let output = run_with_timeout(&mut cmd, QUICK_TIMEOUT).await;
```
