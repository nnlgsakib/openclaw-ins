1. Create `src/components/config/dynamic-config-section.tsx`:

A generic component that renders a config section from `ConfigSectionMeta`:
- Collapsible Card with section label and description
- Renders each field based on its type:
  - `text` → `<input type="text">`
  - `password` → `<input type="password">` with show/hide toggle
  - `select` → `<select>` with options
  - `boolean` → Toggle/Switch component
  - `number` → `<input type="number">`
  - `object` → JSON textarea (for complex nested config)
  - `key-value` → Key-value pair editor
- Advanced fields hidden behind "Show advanced" toggle
- Reads current value from config store using dotted path (e.g., "gateway.port")
- Writes changes back to config store via `updateField(path, value)`
- Validation: required fields highlighted, invalid values show inline error

Props:
```typescript
interface DynamicConfigSectionProps {
  section: ConfigSectionMeta;
  config: Record<string, unknown>;
  onUpdate: (path: string, value: unknown) => void;
}
```
