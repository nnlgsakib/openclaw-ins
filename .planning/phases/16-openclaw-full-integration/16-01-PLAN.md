pub struct ConfigFieldMeta {
    pub key: String,

2. **Tauri commands:**

```rust
#[tauri::command]
pub async fn get_all_channels() -> Result<Vec<ChannelMetadata>, AppError>

#[tauri::command]
pub async fn get_all_providers() -> Result<Vec<ProviderMetadata>, AppError>

#[tauri::command]
pub async fn get_openclaw_metadata() -> Result<OpenClawMetadata, AppError>
```

3. **Implementation approach:** Each command returns hardcoded data extracted from OpenClaw source. This is a static data module — NOT a parser that reads OpenClaw files at runtime. The data is derived from:
   - `openclaw/src/channels/ids.ts` → channel IDs
   - `openclaw/extensions/*/openclaw.plugin.json` → extension metadata
   - `openclaw/src/config/bundled-channel-config-metadata.generated.ts` → channel config fields
   - `openclaw/src/plugins/bundled-plugin-metadata.generated.ts` → provider/auth info

**Channel list to include (all from ids.ts + extensions/):**
- Built-in (9): telegram, whatsapp, discord, irc, googlechat, slack, signal, imessage, line
- Extensions (18+): bluebubbles, feishu, matrix, mattermost, msteams, nextcloud-talk, nostr, synology-chat, tlon, twitch, voice-call, openshell, zalo, zalouser, phone-control, device-pair
