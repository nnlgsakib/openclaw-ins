# Plan 02: Per-Layer Progress Tracking and Animation Enhancement

## Objective

Implement per-layer Docker progress tracking with smooth animations by enhancing the Rust backend to emit layer-specific progress events and creating frontend components to display them. This provides detailed visibility into the Docker image pull process.

**Purpose:** Give users granular insight into installation progress by showing individual layer download status with animated progress bars.

**Output:** A system that tracks and displays real-time progress for each Docker layer during installation, integrated alongside the Docker log viewer.

    app_handle: &tauri::AppHandle,
    step: &str,
    percent: u8,
    message: T,

    tauri::event::emit(
        app_handle,
        "install-progress",
