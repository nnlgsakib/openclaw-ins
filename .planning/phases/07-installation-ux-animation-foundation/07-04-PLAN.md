# Plan 04: Animation Utilities

## Objective

Create a centralized animation utilities file that provides consistent timing, easing, and spring physics presets for use across the application. This ensures animation consistency and makes it easy to update animation values globally.

**Purpose:** Avoid animation duplication and inconsistency by providing a single source of truth for animation values.

**Output:** A reusable animation utilities module with timing presets, spring configurations, and a helper function to generate transition objects.
