## Objective

Enhance the existing shadcn/ui Button component to use Framer Motion for hover, press, and focus micro-interactions. This provides subtle, responsive feedback that improves the user experience without being distracting.

**Purpose:** Replace basic CSS hover effects with physics-based animations for more engaging and polished button interactions.

**Output:** A Button component that uses Framer Motion's whileHover, whileTap, and whileFocus props to animate scale transforms for hover, press, and focus states.
