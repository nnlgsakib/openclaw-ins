## Objective

Upgrade TypeScript from 5.8.3 to 6.0.2 (Dependabot PR #7) and fix any type errors.

**Purpose:** TypeScript 6.0 is a major release with stricter type checking, new features (import defer, pattern narrowing improvements), and deprecations. Dependabot PR #7 proposes this upgrade. Must verify no breaking type changes in the codebase.

**Output:** Updated package.json with `~6.0.2`, fixed type errors, working build.

## Context

@package.json
@tsconfig.json

TypeScript 6.0 key changes:
- Stricter control flow analysis
- `import assert` deprecated (use `import with`)
- Improved pattern matching/narrowing
- Better type inference for generic functions
- Potential new errors from stricter checking

Risk: The `~5.8.3` (tilde) specifier locks to 5.8.x. Changing to `~6.0.2` is a MAJOR bump. Any type errors in the codebase will surface.
