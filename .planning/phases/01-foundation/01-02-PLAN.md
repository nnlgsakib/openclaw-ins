# Plan 02: Foundation — UI Shell + Error Infrastructure

## Objective

Build the application shell with sidebar navigation, implement the structured error system (ERR-01), create page stubs for all 6 routes, and set up the Rust AppState container. This creates the navigable UI framework.

**Purpose:** Deliver a functional app shell where users can navigate between pages and errors display as plain-language messages — the first two success criteria.

**Output:** App launches with sidebar navigation, header, routed pages, and an error infrastructure that maps Rust errors to human-readable toasts.
