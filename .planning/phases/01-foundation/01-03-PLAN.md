# Plan 03: Foundation — Platform Detection + Dashboard + Error Display

## Objective

Implement platform detection using `tauri-plugin-os`, create the PlatformBadge for the header, build the Dashboard page with welcome content, and wire error toast notifications. This completes all three Phase 1 success criteria.

**Purpose:** Deliver the remaining user-facing features: platform awareness (PLAT-01, PLAT-02), error display (ERR-01), and the dashboard landing page.

**Output:** App detects OS, header shows platform badge, dashboard displays welcome content, and errors render as toasts with plain-language messages and fix suggestions.
