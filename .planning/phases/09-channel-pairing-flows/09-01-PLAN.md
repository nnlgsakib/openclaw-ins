# Phase 9: Channel Pairing Flows - Plan

**Created:** 2026-03-26
**Status:** Complete
**Requirements:** CHAN-03, CHAN-04, CHAN-05

## Goal

Implement channel-specific setup wizards (QR, token entry).

## Success Criteria

1. User can pair WhatsApp by scanning QR code displayed in-app
2. User can set up Telegram bot with guided token entry and validation
3. User can set up Discord bot with guided token entry and validation
4. Setup wizards show clear instructions and error handling

## Implementation

- PairingModal: dialog wrapper with channel-specific pairing components
- WhatsApp: QR code display with polling, scan confirmation
- Telegram: Token input with format validation + BotFather link
- Discord: Token input with length validation + Developer Portal link
- Custom Dialog component (no Radix Dialog dependency)
- Tauri commands: get_whatsapp_qr, validate_telegram_token, validate_discord_token
