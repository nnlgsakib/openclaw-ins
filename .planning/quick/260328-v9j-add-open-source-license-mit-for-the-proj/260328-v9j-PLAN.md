# Quick Task 260328-v9j: Add open-source license (MIT)

## Task

Add a LICENSE file and declare the license in package.json and Cargo.toml.

## Approach

The README already references a MIT license badge. MIT is the best fit:
- Maximally permissive (commercial use, modification, distribution, sublicensing)
- Simple and well-understood by the open-source community
- Standard for Tauri desktop apps and developer tools
- Compatible with all project dependencies

## Plan

- `package.json` (add `"license": "MIT"`)
- `src-tauri/Cargo.toml` (add `license = "MIT"`)

**Verify:**
